API Documentation

Welkom bij de API-documentatie. Hieronder vind je een overzicht van alle beschikbare endpoints.

Users

  • Get All Users
    GET /users
    Ophalen van een lijst met alle gebruikers.
  • Get User By ID
    GET /users/:id
    Ophalen van details van een specifieke gebruiker.
  • Create User
    POST /users
    Toevoegen van een nieuwe gebruiker.
    Body: 
    {
    "first_name": "John",
    "last_name": "Doe",
    "email": "john.doe@example.com",
    "password": "password123"
}
  • Update User
    PUT /users/:id
    Updaten van een bestaande gebruiker.
    Body: 
    {
    "first_name": "Jane",
    "last_name": "Smith"
}
  • Delete User
    DELETE /users/:id
    Verwijderen van een gebruiker.

News

  • Get All News
    GET /news?limit=10&offset=0
    Ophalen van een lijst met nieuwsitems.
  • Get News By ID
    GET /news/:id
    Ophalen van details van een specifiek nieuwsitem.
  • Search News
    GET /news/search?title=example
    Zoeken naar nieuwsitems op basis van titel.
  • Create News
    POST /news
    Toevoegen van een nieuw nieuwsitem.
    Body: 
    {
    "user_id": 1,
    "title": "Breaking News",
    "content": "This is the content of the news.",
    "image_url": "http://example.com/image.jpg"
}
  • Update News
    PUT /news/:id
    Updaten van een bestaand nieuwsitem.
    Body: 
    {
    "title": "Updated News Title",
    "content": "Updated content."
}
  • Delete News
    DELETE /news/:id
    Verwijderen van een nieuwsitem.
